# 概要 - 記事の

『UNBEATABLE』はSteamとPS5のプラットフォームに、2025-12-10に発売されたリズムゲーム。

アドベンチャーモードとアーケードモードの二つのモードがあり、それぞれ別途で楽しむことができる。

アドベンチャーモードは "音楽が犯罪とされているが音楽を演る" ストリート・パンク的な世界。

会話の選択肢に遊びが見られ、時々、高品質なアニメーションで物語が表現される。

アーケードモードは "軽快なバンドサウンドやラップ、いかにも音ゲーなEDM remix" などの楽曲が現時点で76曲存在する。

2レーン方式の (Muse Dash的な) 譜面だが、一方向ではなく左右から来るため、実質4レーン相当の認知負荷を要すると考えられる。

VHSのノイズなど世界観を構築するための演出が多数あるが、長時間プレイする場合目が疲れやすい。  
これは設定から切り替えて解消ができる。

こだわり抜かれた演出によってゲームとしての体験の質を高めており、上記の要素の複合が無二性を与えていると感じられた。

定価は3200円だが、Steamにおいては 2025-12-24 まで10%Offで購入できる。

アーケードモードのみの体験版があるため、気になったらまずはそちらを試してみてほしい。

# 概要 - ゲームの

![](/images/unb_main.webp)

2025-12-10 リリース

インストール容量：約32GB

現時点 (2025-12-14) において日本語未対応。タイトルロゴと製品ページ以外は。

ですがなんと……？英語には対応しているため、英語がわかればスムーズに楽しめます！  
日本語は対応予定とのことですが、時期は未定。

さて、このゲームの簡潔な説明はSteam製品ページに求めた方が早いでしょう。

> 『UNBEATABLE』は、音楽が違法な世界で犯罪を冒すリズムアドベンチャーだ。アーケードの完璧なリズムゲームプレイで大きな感動に満ちた物語体験で、ビートと彼女のバンドのストーリーを追いかけよう。
>
> Steam - UNBEATABLE 製品ページ説明より

「アドベンチャーモード」（ストーリー）と「アーケードモード」（リズムゲーム）がそれぞれ独立したものになっています。

そしてアーケードモードで遊べる楽曲は、ストーリーを進める or アーケードモードで解放する、どちらでもすべての楽曲をアンロックできるらしい。

楽曲数は全76曲とのこと。  
さらにSteam製品ページの記述が正しければ、76曲なのは現時点であり、今後も増えるらしい。

またDLCがあるため、それを購入すればもっと増える様子。

# ストーリー

製品ページのおかげで、"音楽が違法な世界での犯罪……音楽やるんだな"ぐらいはわかりますが、どんな感じなのでしょうか。

以下あらすじはゲーム開始30分程度のものです。

![](/images/unb_Eve.webp)

イヴという女性がライブにて、ベーシストとドラマーの二人を待たせているところから始まる。

![](/images/unb_noodle_a_sixteen.webp)

イヴはステージに向かってライブを始めるのだが、その途中で、ピンク色の髪をしたビートという少女に切り替わる。

![](/images/unb_beat.webp)

ビートは、音楽が違法になっているにも関わらず、ギターとアンプを持っていた。

![](/images/unb_qua.webp)

近くにいた少女クェーバーにそれを咎められ、彼女について行くことに。

実はクェーバーはイブの娘らしい。
そして、ビートの持っていたギターを見るなり、ギターを引きたくなったようだ。

かつてイヴがライブを行った会場にて、いざ音楽を奏でると  
音楽を取り締まる組織が現れ、今度は謎のモンスター？が現れ……

逃亡と闘争を繰り返しながら、どうして音楽が犯罪となってしまったのか、その秘密を解き明かしていくことになる。

プレイヤーの操作キャラとしての主人公はビートですが、  
物語の主人公はクェーバーといった印象。

ビートがなぜギターとアンプを持っていたのか、謎。

ボーイ・ミーツ・ガールな感じと逃避行な構造っぽいですが、警官やモンスターを殴る攻撃性もある。  
このねじれが面白そうな予感を醸し出しますね。

# ゲーム

2レーンの音ゲー……そう、『Muse Dash』と言えばわかりやすいでしょうか。

まぁ、ゲームですし動画で見たほうが早いですね。楽曲の雰囲気もかねて。

{{< youtube szLxK7Bxg8A >}}  

{{< youtube d6x2baL_pks>}}

ハイ。かなり演出に凝っています。  
そして何気に1サビ分じゃなくて、フルサイズ。

VHSのようなノイズを載せたり、カメラをズームしたり、  
ここまで映像美にこだわった音ゲーというのも珍しく、本作の特徴でもあります。

さて、操作に必要なボタンは2つだけ、という意味で2レーンではある……のですが、左右に振られることによって認知的には4つに思える。  
これがやってみると意外と難易度を高めているんですよね……

楽曲の方は、バンドサウンドなものがメイン。  
remixや提供楽曲もあるため、いかにも音ゲーっぽい電子サウンドマシマシな曲も。

## 👍️グッド・ポイント

### 圧倒的演出

初回起動や先ほどのアニメーションのように、とにかく演出に強いこだわりを感じる。

{{< youtube ozsvnWokPIA>}}

{{< youtube UgoaqtG22u4>}}

アドベンチャーモード中にも、こんな感じの特殊なリズムゲームが出てくる！

音ゲーによくある譜面のタイプ以外にも、こういった「リズム天国」シリーズのようなゲームがあり、  
音楽やリズムを楽しむのが第一、方法は選ばない（たくさん入れる）という心意気を感じます。

逃げながら犯罪（音楽）を行うという、まぁそれなりにテーマが重いのですが、  
演出や音楽が軽快な雰囲気を作ってくれるので、明るく楽しめます。

### 細やかな難易度

アーケードモードで遊べる楽曲の難易度が細かいのも嬉しいポイント。

![](/images/unb_arcade.webp)

難易度はどの楽曲も分かれており、

- beginner
- normal
- hard
- expert
- unbeatable
- それ以上 （楽曲ごとに命名が異なる。この難易度がない場合も）

このように細分化され、難易度を気にすること無く、曲の好みで選べる。

## 👎️ バッド・ポイント

### 目が疲れる！

肝心のゲーム部分、アーケードモードの演出が疲れます。

（そしてVHSのノイズは動画のエンコード殺し！）

ただ、この問題点は解消できます。

![](/images/unb_options.webp)

設定からVHSフィルターやらなんやら、演出のオン / オフ が切り替えられます。

以下比較。全画面表示で見ると、その差がわかりやすいかと。

VHS と POST PROCCESSING を OFF にしています。

{{< youtube xAe2wDqjj0M>}}

私の場合は、アドベンチャーモード2時間 + アーケードモード2時間  
と合計4時間ぶっ続けで遊んでいたので、それが原因かもしれないですが……

ただ高難易度ほどノーツの情報量が多くなり、視認性の良し悪しが効いてくると思います。  
気になる方はぜひ設定を試してみてください。

### 日本語未対応・時期未定

少なくとも対応予定と発表されているだけ、いいニュース。

翻訳する手段はたくさんあるので、じっくり翻訳しながらストーリーを読み進めるのもありですね。

### アニメ → ゲームの遷移が急すぎる？

パッとすぐに切り替わるため、読み込みのラグを感じない。

……のですが、切り替わりが急すぎて困ることがあります。

それは、アドベンチャーモード中にリズムゲームが出てくるのですが

![](/images/unb_anime2.webp)

リズムゲーム → 途中にアニメーション → リズムゲーム

この遷移が結構急なので、アニメ→リズムゲームの切り替わりでミスしやすい。

そこまで大きな不満ではないですが、小さなストレスです。

いまのところ、反射神経を鍛える以外に対処法はなさそうですね。

アドベンチャーモードのチャプター2の後半はこのゲームの魅力がたっぷり詰まっているので、ぜひ体感してみてほしい……！
